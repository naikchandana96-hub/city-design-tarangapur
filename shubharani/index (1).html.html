Algorithm 1: Round-Robin Call Distribution

üéØ Problem Statement

Given N agents with multilingual capabilities and M incoming calls/tasks in various languages, distribute the workload such that: (1) Each call is assigned to an agent who speaks the required language, (2) Workload is distributed fairly with minimal variance, (3) No agent is overloaded while others are idle, (4) Average wait time is minimized, (5) System operates 24/7 with shift rotations.

Input: Queue of calls with language requirements, List of available agents with language skills
